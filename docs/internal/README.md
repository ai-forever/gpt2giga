# Internal docs

Этот каталог предназначен для internal planning, handoff и task-breakdown документов.

- не входят в canonical user-facing docs surface;
- не должны использоваться как основной source of truth при расхождении с кодом, CI, `README.md` или основными документами из `docs/README.md`;
- могут устаревать быстрее, чем operator-facing документация.
